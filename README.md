# Chrome-os-store
Наконец-то нормальный магазин приложений!
созданый в первую очередь для chrome os подомных систем
(chrome os,chrome os flex,fyde os).

# Что такое chrome os store? (не путать с chrome web store)
Это opensource проект который максимально похож на системный интерфейс chrome os (Material design как на андроид 13+)
этот магазин был создан как красивая альтарнотива gnome-software или kde-discover. Магазин поддерживает Flatpak и appimage
